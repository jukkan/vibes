üéØ Safe Universal Shortcuts

These work in BOTH regular apps and terminals:

  • Ctrl+Insert = Copy (Old-school but reliable)
  • Shift+Insert = Paste (Works everywhere)
  • Shift+Arrow keys = Select text in command prompt

üí° Pro Tips:
  • Mouse selection: In most modern terminals, selecting text with mouse automatically copies it
  • Right-click paste: Works in nearly all terminal windows (safest option)
  • Cancel a command: Ctrl+C (this is its actual job in terminals!)
  • Clear screen: Type cls (CMD/PowerShell) or clear (bash)
  • Autocomplete: Press Tab to complete commands/paths

üìã Quick Reference Card

When in doubt in a terminal:

  1. Copy: Select with mouse ‚Üí Right-click (or Ctrl+Shift+C)
  2. Paste: Right-click (or Ctrl+Shift+V)
  3. Cancel command: Ctrl+C
  4. Never: Use Ctrl+A expecting "select all" or Ctrl+Z expecting "undo"
